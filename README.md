# Agents Go

A simple Go project for experimenting with agent-based systems.

## Features

- Modular agent architecture
- Easy-to-read codebase
- Example agents included

## Getting Started

```bash
git clone https://github.com/yourusername/agents_go.git
cd agents_go
go run main.go
```
